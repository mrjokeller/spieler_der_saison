# VfB STR – Spieler der Saison

Eine visuelle Darstellung der Wahl zum **Spieler der Saison des VfB Stuttgart** vom VfB STR Podcast.
Diese Projekt wurde von einem Fan des VfB STR Podcasts erstellt und steht in keiner offiziellen Verbindung zum Podcast oder zum VfB Stuttgart.

## Technische Umsetzung

### Frontend
- HTML/CSS
- JavaScript
### Backend
- Python
- SQLite
	- `games`: Spielinformationen (Gegner, Ergebnis, Datum, etc.).
	- `player`: Spielerinformationen (Name, Position, Trikotnummer, etc.).
	- `player_data`: Spielspezifische Daten (Minuten, Tore, Vorlagen, etc.).
	- `player_points`: Punktevergabe pro Spieler und Spiel.
